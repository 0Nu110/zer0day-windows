What is Vector Shield?

Vector Shield is a program unlike any other anti-virus out there. Why? Because it isn’t. Vector Shield's job is not to remove viruses, but to defend against them. How? Vector Shield protects your system by securing your computer's settings, disabling services that hackers use to attack your computer, and much more. Don’t worry about these actions hurting your system, it is all configurable.

How Does Vector Shield Work?

Vector Shield uses a series of batch scripts to implement industry standard changes to the system. The user simply opens the application and decides which settings they would like to make more secure.
