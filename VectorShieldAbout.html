About Vector Shield












Vector Shield

Vector Shield is a comprehensive security application built for the Microsoft Windows operating system that is made to prevent the unauthorized exploit of the system and reduce the amount of attack vectors for potential cyber-attacks.

Vector Shield's easy to understand settings allows you to clearly see what is being done to your computer.

Vector shield uses high quality batch scripts to systematically implement industry standard system hardening procedures on the Windows operating system. These procedures are derived from the Centers for Internet Security's annual benchmark, a guideline that our program will use to meet the needs of end users.

The batch script coupled with a Java GUI makes our program highly customizable, and all source code is open source for security auditing and is published under the GNU open source license to insure full transparency.



Cybersecurity

Cybersecurity is important because it protects you and your information from threats that could potentially harm you. Important cybersecurity terms that you should know include:

-Malware: A type of software that can be used to harm a computer. This includes a wide variety of softwares such as worms, trojans, viruses, ransomware, spyware, and adware.

-Exploit: Malicious software that is used to cause harm to or damage a computer.

-Hacker: A person who breaks into computer systems or networks generally in order to obtain unauthorized information.

-System Hardening: The use of good practices and software to remove vulnerablities and make a system more secure.
